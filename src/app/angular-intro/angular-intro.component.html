Angular Project Creation

Commands
  1. Install Node.js in Your System
  2. Install Vs code in Your System
  3. Install Angular CLI by below command
    4. npm install -g @angular/cli
  4. To Create Project by below command
    5. ng new my-app
  5. Change Directory
    6. cd my-app
  6. Run The Project
    7. ng serve
  7. Run The Project And Open Directly in the Browser
    8. ng serve -open
  8. To Create Componenet
    9. ng generate componenet name
  9. To Create Service
    10. ng generate service name
  10. To Create Module
    11. ng generate Module name
